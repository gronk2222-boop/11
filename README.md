# Mentor Bot

Telegram-бот «Наставник» для менеджеров продаж в стройке.

## Возможности
- Конфиденциальный онбординг с шифрованием ответов
- Ежедневная микроцель (без денежных формулировок)
- Режим «Безразличный профессионал» для сбора отказов
- Ритуал «Помидор + Банан» против прокрастинации
- Виртуальный «Добрый директор» и вечерний отчёт
- Follow-up навигатор по клиентам
- Тренажёр первой фразы, чек-лист «Рюкзак разведчика»
- Долговременная память и адаптация через Groq LLM
- Полное удаление данных по команде /forget

## Деплой на Railway
1. Форкни репозиторий.
2. Создай новый проект на Railway, подключи репозиторий.
3. Добавь переменные окружения:
   - `BOT_TOKEN`
   - `DATABASE_URL` (PostgreSQL)
   - `GROQ_API_KEY`
   - `ENCRYPTION_SALT` (любая строка, секретно)
4. Railway автоматически запустит бота.

## Тесты
